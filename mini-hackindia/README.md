# 🤖 Prompt X UI Generator

**A Hybrid AI Framework for Prompt Engineering and Automated UI Design**

---

## 🏁 Pre-HackIndia Virtual Hackathon Submission

This project is submitted as part of the **Pre-HackIndia Virtual Hackathon**, focusing on building a practical, innovative, and deployable AI-based solution.

---

## 🚀 Project Overview

Prompt X UI Generator is a hybrid AI-powered system that enhances user prompts, generates concise and accurate answers, and automatically creates frontend UI designs.  
The project integrates **prompt engineering** with **automated UI generation**, enabling users to quickly transform ideas into functional interfaces.

---

## ❓ Problem Statement

Users often struggle to write effective prompts and convert ideas into usable frontend designs.  
Poorly structured prompts lead to unclear answers, while UI prototyping requires additional technical effort.

---

## 💡 Proposed Solution

The proposed system refines user prompts using AI, provides concise and factual responses, and optionally generates complete frontend UI code (HTML, CSS, JavaScript) with a live preview—bridging the gap between ideas and implementation.

---

## ✨ Key Features

- 🔮 Intelligent prompt enhancement  
- 📘 Concise, AI-generated answers  
- 🎨 Automated frontend UI generation  
- 👀 Live UI preview inside the app  
- ⚡ Fast inference using Groq LLM  
- 🔐 Secure API key management
