# Prompt X — AI-Powered Prompt Enhancer & UI Generator

An AI tool that transforms vague user inputs into optimized, 
production-ready prompts and generates UI mockups instantly.

## 🚀 Features
- Converts basic prompts into detailed, optimized AI prompts
- Generates UI mockup suggestions from natural language
- Real-time response using Groq API (ultra-fast LLM inference)
- Clean, intuitive interface built with Streamlit

## 🛠️ Tech Stack
- Python
- Streamlit
- Groq API
- Prompt Engineering

## 💡 How It Works
1. User enters a basic idea or prompt
2. AI enhances it into a detailed, structured prompt
3. UI mockup suggestions are generated instantly

## 🔧 Setup
```bash
pip install streamlit groq
streamlit run app.py
```
